# -Jharkhand-CivicFix---Crowdsourced-Civic-Issue-Reporting-Resolution-System-SIH25031-
Flutter+Firebase app for SIH25031 (Govt of Jharkhand). Citizens report potholes/garbage with photo+GPS, track status live. Admin dashboard shows analytics, heatmaps, resolution time. SDG9 Clean Green Tech. Features: Real-time tracking, ward analytics, complaint assignment. SIH2025 submission.
